How to Install Your Signature

Step 1: Copy Your Signature

  1. Click your name in the Quick Select grid (or fill in your info manually)
  2. Click the "Copy Signature to Clipboard" button (the big red button)

Step 2: Paste into Outlook

  1. Outlook Desktop (Classic):
    Go to File → Options → Mail → Signatures. Click New, name your signature, then click in the edit box and press Ctrl+V to paste.
  2. Outlook (New / Windows 11):
    Click the gear icon → Accounts → Signatures. Click + New signature, name it, then paste into the editor.
  3. Outlook Web (outlook.com):
    Go to Settings → Mail → Compose and reply. Under Email signature, paste.
  4. Outlook Mobile (iOS / Android):
    Go to Settings → [your account] → Signature. Mobile has limited HTML support — the signature will appear simplified.
Pro Tips:
• Fonts: Montserrat renders in webmail & mobile. Classic Outlook desktop falls back to Arial — this is normal.
• Animated GIFs: The V logo and banner will animate in webmail & mobile. Classic Outlook desktop shows only the first frame.
• "Copy Signature to Clipboard" is the easiest method. If it doesn't paste cleanly, try "Copy HTML" and paste into your signature editor's source/code view.
• Ensure images are hosted at permanent public URLs — if the host goes down, images break.
